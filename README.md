# Blog-app

The Budget app is a mobile application, where we can manage your budget: we have a list of transactions associated with a category, so that we can see how much money you spent and on what.
## Built With

- Ruby
- Ruby on Rails

## Getting Started

To get a local copy up and running follow these simple example steps.

git clone https://github.com/Somdotta07/budget-tracker.git

cd budget-tracker

### Prerequisites

Ruby
Rails
PostgresSQL

### Setup

Install gems with:

```
bundle install
```

Setup database with:

```
rails db:create
rails db:migrate
```

### Usage

Start server with:

```
rails server
```

Open http://localhost:3000/ in your browser.

### Run tests

Install npm with:

```
npm i
```

Install rspec with:

```
bundle install
```

and

```
rails generate rspec:install
```

run the test with:

```
rspec spec
```
